## What's Being Published

### Core Library (8,613 lines of code)
1. **Explicit Configuration Layering** - ConfigLayer enum + LayerBuilder
2. **Multi-Format Support** - YAML, TOML, JSON, RON, HJSON, JSON5
3. **Bidirectional Editing** - Write-back with TOML comment preservation
4. **Metadata & Introspection** - Full schema inspection and validation API
5. **Multi-Scope Resolution** - Platform-appropriate paths (user/project/system)
6. **Provenance Tracking** - Know where each config value originated
7. **Database Support** - SQLx integration (feature: database)
8. **HTTP Support** - URL validation (feature: http)
9. **Validation Framework** - Comprehensive constraint-based validation
