Users upgrading from 0.15.0 should:
1. Update `Cargo.toml` dependency to `settings_loader = "1.0"`
2. Optionally adopt new `LayerBuilder` for explicit configuration
3. Continue using existing `SettingsLoader` trait (backward compatible)

**No breaking changes to existing code required.**
